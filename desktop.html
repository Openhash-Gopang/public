핵심 특징

AI 공무원은 어떻게 다른가요

단순 민원 앱이 아닙니다. 현실 정부를 대신해 실제 업무를 수행하는 AI 행정 시스템입니다.

K-Public 핵심 우위

광범위한 행정 지식 + 전 생애 PDV 기록

K-Public AI 공무원은 두 가지를 동시에 갖춥니다. 첫째, 전국의 법령·조례·행정 절차를 웹 검색과 공공 데이터 포털을 통해 실시간으로 파악합니다. 어떤 담당 공무원도 모든 법령과 조례를 혼자 알 수는 없지만, AI 공무원은 즉시 확인합니다. 둘째, 시민의 고팡 PDV에 저장된 과거 행정 처리 이력·가족 정보·주소 이력을 동의 하에 참조하여 서류를 자동 작성합니다.

시민은 복잡한 서류를 채울 필요가 없습니다. 요청만 하면 됩니다. 이것이 K-Public만이 제공할 수 있는 고유한 가치입니다.

📜 법령·조례 실시간 참조 🔐 PDV 기반 서류 자동 작성 📊 공공 데이터 포털 연동 🏛️ 전국 행정 기관 연계

민원 서류 자동 발급

등본·초본·인감·사업자등록증·납세증명 등 모든 민원 서류를 AI가 대신 신청·발급합니다. PDV에 저장된 정보로 서류를 자동 작성하며, 발급된 서류는 PDV에 보관됩니다.

📋 정부24·민원24 연계 · PDV 자동 작성 · 발급 즉시 PDV 저장

인허가 신청 대행

건축 허가·식품 영업 허가·각종 신고를 AI가 대신 신청합니다. 필요 서류를 자동으로 파악하고 작성·제출하며, 처리 현황을 실시간으로 추적합니다.

🏗️ 건축법·식품위생법·각종 특별법 실시간 참조 · 필요 서류 자동 파악

지역 정보 실시간 조회

우리 동네 올해 공공 사업·예산·도로 정비 계획·가로수 현황을 공공 데이터 포털과 웹 검색으로 실시간 조회·요약합니다. 지역 의정 활동·조례도 쉬운 말로 안내합니다.

📊 공공 데이터 포털 · 지방재정 공개 시스템 · 국토정보 플랫폼 연동

공공 시설 신고 & 추적

도로 파손·상하수도 문제·가로등 고장·불법 주정차를 신고하면 담당 부서가 즉시 배정됩니다. 처리 현황을 실시간으로 추적하고 완료 시 PDV에 기록됩니다.

🔧 담당 부서 자동 배정 · 실시간 처리 추적 · 미처리 시 자동 에스컬레이션

맞춤 복지 자동 안내

PDV의 가족 정보·소득·주소를 분석하여 신청 가능한 복지·지원금을 자동으로 찾아줍니다. 신청 서류를 대신 작성하고 제출까지 대행합니다.

💰 복지로·사회보장정보원 연계 · 맞춤 혜택 자동 탐색 · 신청 대행

행정 기록 PDV 보관

모든 민원 처리 결과·발급 서류·행정 이력은 시민 본인의 PDV에 암호화 저장됩니다. 동일 민원 재신청 시 이전 기록을 자동 참조하며, 행정 투명성을 보장합니다.

🔐 AES-256-GCM 암호화 · OpenHash 불변 저장 · 전자정부법 준수
작동 방식

K-Public은 어떻게 작동하나요?

6단계 AI 행정 워크플로우 — 요청부터 결과 기록까지 자동화됩니다.

1

요청 입력

말 또는 텍스트로 요청합니다. "등본 발급해 줘", "건축 허가 받아 줘", "우리 동네 올해 예산 알려줘" — 어떤 형식이든 됩니다. AI 공무원이 요청 유형을 자동 분류합니다.

자연어 요청 · 24시간 응답 · PDV 이력 자동 참조
2

담당 기관 및 절차 파악

웹 검색과 공공 데이터 포털을 통해 담당 기관·부서·최신 절차·필요 서류·처리 기간·수수료를 실시간으로 확인합니다. 법령이 개정되어도 항상 최신 정보를 제공합니다.

웹 검색 기반 최신 절차 · 법령 실시간 확인 · 담당 부서 자동 파악
3

서류 자동 작성

PDV에 저장된 시민 정보(주소·가족·소득·과거 이력)를 동의 하에 참조하여 필요 서류를 자동으로 작성합니다. 시민은 내용을 확인하고 동의만 하면 됩니다.

PDV 정보 자동 반영 · 시민 확인·동의 후 제출 · 오류 자동 검토
4

기관 제출 및 추적

확인된 서류를 해당 기관에 전자 제출합니다. 접수 번호를 받아 처리 현황을 실시간으로 추적하며, 단계별로 시민에게 보고합니다. 처리가 지연되면 자동으로 담당자에게 확인 요청합니다.

전자 제출 · 실시간 처리 추적 · 지연 시 자동 에스컬레이션
5

결과 보고

처리가 완료되면 결과를 시민에게 즉시 보고합니다. 발급된 서류·승인 결과·처리 내역을 정리하여 전달하며, 추가 조치가 필요한 경우 다음 단계를 안내합니다.

완료 즉시 보고 · 서류 전달 · 추가 조치 안내
6

PDV 행정 기록 저장

모든 처리 결과·발급 서류·행정 이력이 시민의 PDV에 암호화 저장됩니다. 향후 동일 민원 재신청 시 자동으로 참조되며, 행정 처리의 전 과정이 투명하게 기록됩니다.

전 과정 PDV 암호화 저장 · 재신청 시 자동 참조 · 행정 투명성 보장
